These are public-safe exports generated by `chatgpt-thread-exporter` from intentionally non-private shared ChatGPT conversations. They exist as stable examples for documentation, blog posts, and visual QA.

## Regenerating

Run the exporter with `--out` and `--force` for Markdown examples, and add `--format pdf` for PDF examples.

Example:

```bash
npm run dev -- --url "https://chatgpt.com/share/..." --out examples/example-name.md --force
npm run dev -- --url "https://chatgpt.com/share/..." --format pdf --out examples/example-name.pdf --force
```
